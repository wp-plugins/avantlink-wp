== Description ==

The AvantLink WP plugin allows quick and simple integration with a variety of affiliate marketing tools from the AvantLink.com network.

Users must have a valid, active affiliate account with AvantLink.com to utilize this plugin.

Once activated, affiliates can quickly and easily use the following tools:
* <a href="http://www.avantlink.com/affiliates/affiliate-tool-center/">Ad Campaigns</a>
* <a href="http://www.avantlink.com/affiliates/affiliate-tool-center/affiliate-link-encoder/">Affiliate Link Encoder</a>
* <a href="http://www.avantlink.com/affiliates/affiliate-tool-center/custom-link-builder/">Custom Links</a>
* <a href="http://www.avantlink.com/affiliates/affiliate-tool-center/product-ad-widget/">Product Ad Widgets</a>
* <a href="http://www.avantlink.com/affiliates/affiliate-tool-center/avantlink-api/">Product Search API</a>

== Installation ==

1. Upload the `avantlink-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Navigate to the `AvantLink WP` settings page in WordPress and enter your AvantLink.com affiliate identification information to enable the plugin.
